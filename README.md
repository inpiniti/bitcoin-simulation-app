# Bitcoin Simulation App

Bitcoin 자동매매 시뮬레이션 앱 (Expo Web).

## 주요 기능
- 서버 상태 모니터링
- AI 질문 (Gemini)
- 딥러닝 스튜디오
- 자동매매 설정
- 포트폴리오 자산 배분 제안
